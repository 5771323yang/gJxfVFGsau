# 项目简介

## 核心功能
- 商品管理
- 分类管理
- 用户管理
- 日志管理
- 系统信息
- 批量删除
- 分页查询

## 适用场景
- 用于管理Python项目的仓库
- 适用于需要跟踪和管理大量Python源码和依赖项的环境
- 适合开发团队或个人开发者进行项目管理

## 技术特点
- 采用Django框架构建，确保项目的高效性和可扩展性
- 使用MySQL数据库存储数据，保证数据的安全性和稳定性
- 前端界面由Vue.js实现，提供友好的用户体验
